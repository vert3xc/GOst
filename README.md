# GOST-Crypto — библиотека и CLI для российских криптографических стандартов  

---

**GOST-Crypto** — это реализация российских криптографических стандартов на языке Go, включающая как библиотеку, так и консольную утилиту (CLI).

Проект предназначен для:
- изучения российских криптографических алгоритмов;
- экспериментирования с ГОСТ-стандартами;
- использования в образовательных и исследовательских целях;
- интеграции в другие Go-проекты.

### Реализованные алгоритмы

#### Хэш-функции:
- **Стрибог (ГОСТ Р 34.11-2012)**  
  - 256 бит  
  - 512 бит  

#### Блочные шифры:
- **Магма (ГОСТ 28147-89)** — 64-битный блок, 256-битный ключ  
- **Кузнечик (ГОСТ Р 34.12-2015)** — 128-битный блок, 256-битный ключ  

#### Цифровая подпись:
- **ГОСТ Р 34.10-2012**
  - кривые:
    - 256a  
    - 512a  
    - 512b  
    - 512c  
    - test  

#### Генератор ПСЧ:
- **ГОСТ Р 1323565.1.006-2017** — детерминированный ГПСЧ на основе хэш-функции.

## Установка и сборка
go install: 
```
go install github.com/vert3xc/GOst/gost-cli@latest
cd go/bin && chmod +x ./gost-crypto && ./gost-crypto
```
Сборка из исходников
```
git clone https://github.com/vert3xc/GOst.git
cd GOst/gost-cli && go build . && chmod +x ./gost-crypto && ./gost-crypto
```

## Использование в качестве библиотеки
```
go get github.com/vert3xc/GOst
```
В коде:
```
import (
  ...

  "github.com/vert3xc/GOst/streebog"
)

func main() {
  hsh, _ := streebog.New(512)
  ...
}
```
